# Home Network Optimization & Router Configuration

## Objective
Optimize home network for VPN and server reliability.

## Problems Encountered
- DHCP reservation confusion
- Potential routing conflicts between router and VPN

## Troubleshooting Process
- Identified need for static IP assignments
- Mapped MAC address to fixed IP
- Separated router and server responsibilities

## Solutions
- Configured DHCP reservation on router
- Ensured consistent internal IP for server
- Stabilized network environment

## Result
- Predictable and stable network
- Reduced connectivity issues
- Improved system reliability

## Time to Resolution
- 30–60 minutes
